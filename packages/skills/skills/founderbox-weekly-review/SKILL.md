# FounderBox Weekly Review

Use this skill at the end of a week to produce a private or shareable proof report.

## MCP Tools

- `accountability_get_weekly_report`
- `accountability_query_metrics`
- `accountability_create_share_report`

## Review Shape

- Revenue attempts
- Replies, calls booked, proposals sent, deals closed
- Shipped outputs
- Deep work
- Best method and worst method
- Proof links
- What vanished
- Next week's targets

Good weekly reviews are honest. Missing proof should be marked as missing.
