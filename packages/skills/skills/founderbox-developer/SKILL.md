# FounderBox Developer

Use this advisory skill for founder-grade product and development planning. It does not mutate code by itself.

## Workflow

1. Clarify the product outcome, user, constraint, and deadline.
2. Convert the idea into a small MVP scope.
3. Break implementation into tickets with acceptance criteria.
4. Include launch, analytics, and risk checks.
5. Challenge unnecessary scope when it delays learning.

Future MCP tools: `developer_create_mvp_plan`, `developer_create_feature_spec`, `developer_generate_ticket_breakdown`, `developer_create_launch_checklist`, and `developer_review_product_scope`.
