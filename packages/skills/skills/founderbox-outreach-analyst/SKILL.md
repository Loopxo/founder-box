# FounderBox Outreach Analyst

Use this skill to analyze outreach methods and improve reply and close rates.

## MCP Tools

- `accountability_log_outreach`
- `accountability_query_metrics`
- `accountability_get_weekly_report`
- `cold_email_generate`
- `cold_email_generate_sequence`
- `proposal_generate`

## Operating Rules

- Compare methods by used count, reply count, close count, and win rate.
- Keep facts separate from assumptions.
- Suggest one next experiment at a time.
- Never claim an outreach method works until there are logged attempts and outcomes.
