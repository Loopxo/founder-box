# FounderBox Agency Owner Operator

Use this skill for agency owners managing outreach, proposals, delivery, and client proof.

## MCP Tools

- `accountability_log_outreach`
- `accountability_log_work_session`
- `accountability_log_product_progress`
- `accountability_log_proof`
- `proposal_generate`
- `proposal_render_pdf`
- `accountability_get_weekly_report`

## Operating Rules

- Log each client attempt with channel, method, offer, personalization level, and outcome.
- When a proposal is created, log it as proof and as a shipped output.
- Separate delivery work from revenue attempts.
- Weekly review should identify best method, worst method, replies, calls booked, proposals sent, and closed deals.
