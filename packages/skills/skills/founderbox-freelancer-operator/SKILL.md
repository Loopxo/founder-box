# FounderBox Freelancer Operator

Use this skill for freelancers balancing client acquisition, billable delivery, follow-ups, and proof.

## MCP Tools

- `accountability_start_day`
- `accountability_log_outreach`
- `accountability_log_work_session`
- `accountability_log_proof`
- `invoice_create`
- `invoice_render_pdf`
- `accountability_end_day_review`

## Operating Rules

- Treat follow-ups as first-class revenue attempts.
- Every work session should name the client/project, minutes, output, and proof.
- If the user logs billable work without proof, ask for a link, screenshot, file name, or short text proof.
- End-day review should include next follow-up and next delivery action.
